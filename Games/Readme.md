# Arcade Games Collection

This folder contains simple arcade-style games built using **C++** and **SDL**, created to explore game loops, real-time input handling, collision mechanics, and 2D rendering fundamentals.

---

## Games Included

### **Snake**
A classic Snake implementation featuring smooth grid-based movement, food generation, scoring, and collision detection.

### **Block Breaker**
An arcade-style block breaking game showcasing paddle controls, ball physics and brick collision logic.

---

## Tech Stack
- **Language:** C++  
- **Graphics & Input:** SDL2 
- **Tools:** VSCode, MinGW

---

## How to Run
- There is no specific make file here. Include path as arguments while compiling.
- Include **-lSDL2main -lSDL2 -lSDL2_ttf** while compiling.
- This paths vary according to your machine and where your **SDL2** is installed, make sure u entered correct path.
